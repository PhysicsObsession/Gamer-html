Welcome to Gamer.html

A hub for every gamer

Information

Gamer.html is a html program made for gamers and by gamers. It is a program used to play and upload games which can be all be played in one place. It is not a spyware or counterfiet website like others in the space.

Navigation

  1. Click on a genre of game that you want to play from the 'Navigation Pane' present towards the left of the screen.
  2. In the navigation pane, images and titles of games will appear from which you can select the game of your choice.
  3. After choosing the game the game will open with the needed information on controls present in this window.
